Quick Introduction

This is a light weight jQuery plugin to create diverse corner effects,such as scoop,folded and so on.

It just uses some pure CSS3 tricks without no images.

If you are interested in how to do this,you can read the book of CSS Secrets by Lea Verou which enlighten me to make it

Star

Getting Started

Step one:
You need to import jQuery and Corners.js like this:

<script src="http://code.jquery.com/jquery-1.10.2.min.js"></script> <script src="Corners.js" type="text/javascript"></script>

Step two:
Suppose that you need to add a kind of corner to the div element which id is corner in the html:

<div id="corner"></div>

Just use it this way in your js:

$('#corner').Corners();

Documentation

Here are some parameters to set and which are not compulsory.Default values are as follows:

extents:15px angle:30 style:beveled defaults:3

style: <String>
As it's name suggests,the style means you can choose what kind of corner you want.

size: <Integer>
without unit.
When you want a folded corner, you can decide how big it is.Tips:the unit is designed em.

angle: <Integer>
without unit.
When you want a folded corner,you can set an angle to show how to fold the element.

extents: <String>
with the unit 'px'
.When you want a beveled or scoop Corner.Not only you can decide how big it is.

Examples

<div id="folded_a"></div> $('#folded_a').Corners({ style:'folded', angle:'50' })
folded
<div id="folded_b"></div> $('#folded_b').Corners({ type:'scoop' })
scoop
<div id="folded_c"></div> $('#folded_c').Corners({ type:'beveled', extent:'30px' })
beveled
